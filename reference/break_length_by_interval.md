# break the given length into several intervals with the given intervals

break the given length into several intervals with the given intervals

## Usage

``` r
break_length_by_interval(length, interval)
```

## Arguments

- length::

  the length to be broken

- interval::

  the given intervals

## Value

: a vector of length of each interval

## Examples

``` r
# example code
 break_length_by_interval(100, 20:26)
#> [1] 20 21 22 23 14
```
